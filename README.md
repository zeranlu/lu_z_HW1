# Welcome to my SWAPI Content Loader

Welcome to the SWAPI (Star Wars API) content viewer.

All the content that you see is dynamically populated using javascript objects from the free database: https://swapi.info/. A fetch request is made in Javascript from URLs such as: https://swapi.info/people or https://swapi.info/films. From these array, specific attributes such as "title" or "name" are defined and iterated within Javascript.
